# simple-database-cpp
Save and load list of integers using file streams


### How to use?
```sh
# Clone repository
git clone git@github.com:trunc8/simple-database-cpp.git
# Change directory
cd simple-database-cpp
# Compile and execute
g++ main.cpp && ./a.out
```
